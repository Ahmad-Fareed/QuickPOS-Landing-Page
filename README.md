> The Last POS System You'll Ever Need. A fully responsive, modern landing page built with vanilla web technologies and a functional PHP backend.

This project was developed to simulate a complete Agile software development lifecycle for a Software Project Management (SPM) course at FAST-NUCES. It demonstrates not just front-end and back-end coding, but strict adherence to industry-standard CI/CD and project tracking workflows.

## ✨ Key Features

* **Responsive UI/UX:** Fully fluid design using CSS Flexbox and Grid, scaling perfectly from mobile to desktop.
* **Vanilla Stack:** Built entirely without external frameworks (No Bootstrap/Tailwind) to demonstrate core competency in HTML5, CSS3, and JavaScript.
* **Interactive Modal:** Custom-built sign-up modal triggered via Vanilla JS event listeners.
* **Secure Backend:** Functional Contact Us form processed via PHP, featuring server-side sanitization (`htmlspecialchars`).

## 📈 Agile Methodology & Workflow

This project was executed over two defined Sprints using a strict Agile framework:

1.  **Jira Issue Tracking:** All tasks were broken down into Epics (Navigation, Hero, Features, Pricing, Backend) and tracked via a Scrum board.
2.  **Git Feature Branching:** Every feature was developed on an isolated branch (e.g., `feature/SCRUM-13-navbar`).
3.  **Pull Request Reviews:** Code was merged to `main` only after passing a two-person review process (Tech Lead & Project Manager).
4.  **Slack Integration:** Real-time CI/CD pipeline notifications were routed to a dedicated Slack channel for transparent team communication.

## 🚀 Sprints Overview

* **Sprint 1 (UI/UX Engineering):** Completed the structural layout, responsive CSS, interactive features, and Bug logging/resolution.
* **Sprint 2 (Backend Logic):** Implemented the PHP contact form and JavaScript modal functionality.
